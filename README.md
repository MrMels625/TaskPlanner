# Планировщик учебных задач

Desktop-приложение для управления учебными задачами, дедлайнами и приоритетами. Разработано в рамках дисциплины **Основы проектной деятельности** в ИКНК СПбПУ.

## Для кого
В первую очередь приложение адресовано **студентам**, которым нужно систематизировать учебную нагрузку, фиксировать сроки сдачи и не терять задачи в потоке информации.

## Что умеет
- Создание, редактирование и удаление задач
- Указание дисциплины, дедлайна, приоритета и описания
- Фильтрация: все / на сегодня / просроченные, а также по приоритету
- Сортировка по дате, приоритету и статусу выполнения
- Блок статистики и автоматическое сохранение в JSON
- ...
